1.We got inspired by PyImageSearch who produced a great tutorial on how to build a face mask detector running on a webcam.

2.So we wanted to take to the next level and use Computer Vision to detect face masks in street cameras video streams.

3.This tool detects people passing by security-like cameras, identifies who is using a face mask, and collects statistics about its usage.

4.The results show that only half of the people passing by are using a face mask.













































1.Given the limited testing facilities for COVID19, there is a rush to develop AI tools for quick analysis using x-rays.

2.AI tools are widely used for medical image diagnosis. The researchers pooled the data of X-ray images of COVID-19 infected patients as well as healthy persons from different sources available on the internet.

3. The deep learning method has the advantage that it learns the disease diagnosing features from the x-ray images in an automatic way..

4.This tool out-performs other such high-tech tools available globally. Moreover, it uses simple machine learning architecture, which makes it stand out over others.
